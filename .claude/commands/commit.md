# /commit

Commit and push all staged and unstaged changes to the remote repository.

## Instructions

1. Run `git status` to see all changes
2. Run `git diff` to understand what changed
3. Stage all changed files with `git add`
4. Create a concise commit message based on the changes
5. Commit with the message
6. Push to the remote repository
7. Confirm success to the user
8. Don't add "Claude" as co-author

Always push automatically - do not ask for confirmation.
